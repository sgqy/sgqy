### 答《赤壁懷苦》

回首身居檣櫓上
\
軟甲布衣盾如綢
\
豪傑談笑功名授
\
煙滅灰飛將帥酬

</div>

<div lang="ja">

### 『赤壁懐古』への返答

首を回して身が檣櫓に居ると気付け
\
軟い甲と布の衣で盾が綢作りの如く
\
豪傑は談笑しながら功名を授けられ
\
煙と滅し灰と飛ばれ將帥のみ酬われ

</div>

<div lang="ja">

### 由来

偉人が功名を立てることについ盛り上がっちゃうだろうが、
\
兵士の自分は簡単に犠牲されることは決して忘れないこと。
